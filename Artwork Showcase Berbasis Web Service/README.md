Judul Project 	: Artwork Showcase Berbasis Web Service
Kelompok      	: Ahdiat Ahsan dan Valendion Pradana P.

Deskripsi		: 
Project ini merupakan sebuah artwork showcase, Sebuah artwork showcase merupakan 
salah satu konsep media atau aplikasi yang sering digunakan untuk berbagi media gambar 
antar sesama organisasi atau komunitas yang kemudian media gambar tersebut diberikan 
tanggapan yang biasanya berupa jumlah like atau suka. Dengan adanya keberadaan sebuah 
artwork showcase akan sangat bermanfaat bagi si pemilik karya dan orang-orang 
yang menyukai hal terkait.
